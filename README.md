# HUC---internship

Git -
Git is a distributed version control system used to track changes in files and manage different versions of a project allowing individuals and teams to collaborate efficiently.

Github -
GitHub is a web-based platform that hosts Git repositories and provides tools for storing code collaborating with others, tracking issues, and managing projects online.

Uses of Git and Github :

Uses of Git:

1.Track changes in code over time

2.Go back to previous versions if errors occur

3.Work on multiple features using branches

Uses of GitHub:

1.Store Git repositories online (cloud backup)

2.Share code with others

3.Collaborate with teams on the same project

Basic Git Workflow -

1.Commit Changes
Save changes with a message.

2.Create and Switch Branches
Work on features separately.

3.Merge Branches
Combine changes from different branches.

4.Pull Changes
Get updates from remote repository.

5.Push Changes
Upload commits to GitHub.

6.Git push
pushing the code to github

How Github helps in collaboration :

GitHub helps in collaboration by allowing multiple people to work on the same project safely and efficiently
It enables team members to use branches, pull requests, code reviews, and issue tracking.
